# Ngaji Yuk! 🌙

Belajar mengaji jadi seru dan interaktif untuk anak-anak Indonesia.

## 🎯 Fitur
- **Level 1:** Huruf Hijaiyah + Makhraj (✅ Selesai - 30 VO)
- **Level 2:** Makhraj (✅ Selesai)
- **Level 3:** Harakat (🔜 VO sedang dibuat)
- **Level 4:** Talaqqi AI (🔜 Segera)
- **Level 5:** Ilmu Tajwid (🔜 Segera)

## 🚀 Cara Deploy

### GitHub Pages
1. Settings → Pages → Source: GitHub Actions
2. Atau: push ke branch `main`, lalu setting Pages ke root folder

### Akses
Landing page bisa diakses via: `https://ganststudio-coder.github.io/ngaji-yuk-landing/`

## 🎙️ VO
File audio MP3 ada di folder `audio/`. Abah bisa generate VO baru dan replace di folder ini.
